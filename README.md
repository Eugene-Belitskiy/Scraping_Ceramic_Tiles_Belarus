# Scraping Belarus — Мониторинг рынка керамической плитки

## Описание проекта

**Scraping Belarus** — система для сбора, обработки и анализа данных о керамической плитке на белорусском рынке. Проект предназначен для оперативного мониторинга конкурентной среды и определения позиции продукции **ОАО "КЕРАМИН"**.

### Ключевые возможности

- **Автоматический парсинг** данных с белорусских ритейлеров (Phase 1: 3 магазина)
- **Гармонизация данных** — унификация цветов, дизайнов, брендов, типов поверхности
- **Два хранилища данных** — `products.json` (товары) и `prices.json` (цены по месяцам)
- **Supabase** — облачная PostgreSQL-база для хранения и доступа к данным
- **Streamlit-дашборд** — интерактивный анализ рынка с фильтрами и визуализацией

## Установка и настройка

### Требования

- Python 3.8+
- Google Chrome (для Selenium)
- Аккаунт [Supabase](https://supabase.com)

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Настройка Supabase (один раз)

1. Создать проект в [Supabase](https://supabase.com)
2. В SQL Editor применить схему из `dashboard/create_tables_v2.sql`
3. Создать файл `dashboard/.env`:

```
SUPABASE_URL=https://xxx.supabase.co
SUPABASE_KEY=your-anon-key
```

## Ежемесячный рабочий процесс

### 1. Парсинг источников

```bash
# Phase 1 — три магазина
python Altagamma/Altagamma.py
python 21vek/21vek_request.py
python Modus_Keramica/Modus.py
```

Результаты сохраняются в JSON-файлы в папках соответствующих магазинов.

### 2. Объединение и гармонизация

```bash
python MERGED_BELARUS/Main_scraping_Belarus.py
```

Скрипт:
1. Загружает данные из всех источников
2. Фильтрует по материалу (Керамогранит, Керамика, Клинкер)
3. Нормализует форматы плитки
4. Рассчитывает диапазоны цен и скидок (шаг 10 BYN)
5. Применяет гармонизацию данных
6. Удаляет дубликаты
7. Сохраняет `MERGED_BELARUS/data_finally.json`

### 3. Разбивка на две таблицы

```bash
python MERGED_BELARUS/migrate_to_two_tables.py
```

Создаёт:
- `MERGED_BELARUS/products.json` — уникальные товары (product_id = md5(url)[:16])
- `MERGED_BELARUS/prices.json` — записи цен по дате (price_id = {product_id}_{date})

### 4. Загрузка в Supabase

```bash
python dashboard/upload_to_supabase.py
```

Выполняет upsert по первичному ключу — безопасно перезапускать.

### 5. Запуск дашборда

```bash
streamlit run dashboard/dashboard.py
```

Локально откроется по адресу `http://localhost:8501`

## Структура данных

### Таблица `products` (уникальные товары)

| Поле | Описание |
|------|----------|
| `product_id` | MD5(url)[:16] — стабильный идентификатор |
| `name` | Полное наименование |
| `url` | Ссылка на товар |
| `store` | Магазин |
| `material` | Материал (`Керамогранит`, `Керамика`, `Клинкер`) |
| `brand` | Бренд (ВЕРХНИЙ РЕГИСТР) |
| `country` | Страна производства |
| `brand_country` | Комбинация "БРЕНД (СТРАНА)" |
| `format` | Нормализованный формат ("60x60") |
| `color` / `primary_color` | Цвет (полный / основной) |
| `design` / `primary_design` | Дизайн (полный / основной) |
| `surface_type` | Тип поверхности (`Матовая`, `Глянцевая`, ...) |
| `thickness` | Толщина, мм |
| `collection` | Коллекция |

### Таблица `prices` (цены по месяцам)

| Поле | Описание |
|------|----------|
| `price_id` | `{product_id}_{date}` |
| `product_id` | FK → products |
| `price` | Цена (BYN) |
| `price_range` | Диапазон цены (шаг 10 BYN, напр. "20-30") |
| `price_unit` | Единица (`м²`, `шт.`) |
| `discount` | Скидка (%) |
| `availability` | Наличие (`В наличии`, `Под заказ`, `Нет в наличии`) |
| `date` | Дата мониторинга |

### Представление `tiles_v2`

JOIN products + последние цены — используется дашбордом.

## Модуль гармонизации

[`MERGED_BELARUS/harmonization.py`](MERGED_BELARUS/harmonization.py) приводит значения к единому стандарту:

| Поле | Результат |
|------|-----------|
| Единицы измерения | `м²`, `шт.`, `упаковка` |
| Дизайн | Синонимы объединены, комбинации сохранены (`"Мрамор, Камень"`) |
| Цвет | Оттенки убраны (`"светло-серый"` → `"Серый"`), комбинации сохранены |
| Тип поверхности | Женский род, капитализация (`"матовый"` → `"Матовая"`) |
| Бренды | Суффиксы удалены (`CERAMICA`, `CERAMICHE` и др.) |
| Наличие | `"Остаток X м2"` → `"В наличии"`; `"Предзаказ"` → `"Под заказ"` |

## Дашборд

Интерактивный Streamlit-дашборд с 5 вкладками:

1. **Ценовой ландшафт** — скрипичные диаграммы по сегментам конкурентов
2. **Угрозы КЕРАМИН** — ценовые диапазоны, в которых конкуренты давят на КЕРАМИН
3. **Позиция по форматам** — пузырьковая диаграмма формат × цена
4. **Поиск аналогов** — пересечение форматов и ценовых диапазонов
5. **Данные** — полная таблица с экспортом в Excel

Конкурентные сегменты: **КЕРАМИН**, Россия (премиум), Россия (прочие), Беларусь, Турция, Индия, Китай, Средняя Азия, Европа, Прочие.
